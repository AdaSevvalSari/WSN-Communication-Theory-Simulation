# 📡 Wireless Sensor Networks — Communication Theory Simulation

This project presents a communication theory analysis of Wireless Sensor Networks (WSNs), focusing on modulation performance, signal behavior, and energy–reliability trade-offs.

---

## 📊 Overview

This project combines theoretical analysis and simulation to demonstrate how core communication theory concepts directly influence real-world wireless system design.

The study is based on IEEE 802.15.4 and highlights the relationship between reliability, range, and energy efficiency.
